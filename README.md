# Website

The LiDiSoft website — a static site served via GitHub Pages at [lidisoft.com](https://lidisoft.com).

## Structure

| File | Purpose |
| --- | --- |
| `index.html` | Landing page — services, values, open-source model, contact. |
| `logo.png` | Brand logo (also used as favicon). |
| `CNAME` | Custom domain configuration for GitHub Pages. |

Everything is self-contained: styles are inlined in each page and the only external
dependency is the Google Fonts stylesheet. There is no build step — edit the HTML and
push.

## Design

The visual language is built from the logo's four shapes — a blue square, a coral
circle, a yellow triangle and a green square — used as the accent palette across the
site.

| Role | Colour |
| --- | --- |
| Blue | `#4a9fe3` |
| Coral | `#f56f5c` |
| Yellow | `#f5cf2b` |
| Green | `#3aa757` |
